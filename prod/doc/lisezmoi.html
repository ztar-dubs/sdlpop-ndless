Prince of Persia pour TI-Nspire CX

SDLPoP-Ndless — Portage de Prince of Persia (1989) pour les calculatrices TI-Nspire CX.

Installation

Configuration requise

  • Calculatrice TI-Nspire CX, CX CAS, CX II ou CX II CAS
  • Ndless installé sur la calculatrice
  • Fichiers de données du jeu original Prince of Persia DOS (fichiers .DAT)

Arborescence des fichiers

Copiez les fichiers suivants sur votre calculatrice. L'exécutable et le dossier data/ doivent être dans le même répertoire.

votre_dossier/ |-- prince.tns (exécutable du jeu) +-- data/ |-- CDUNGEON.DAT.tns |-- CPALACE.DAT.tns |-- EDUNGEON.DAT.tns |-- EPALACE.DAT.tns |-- FAT.DAT.tns |-- GUARD.DAT.tns |-- GUARD1.DAT.tns |-- GUARD2.DAT.tns |-- KID.DAT.tns |-- LEVELS.DAT.tns |-- PRINCE.DAT.tns |-- PV.DAT.tns |-- SHADOW.DAT.tns |-- SKEL.DAT.tns |-- TITLE.DAT.tns |-- VDUNGEON.DAT.tns |-- VIZIER.DAT.tns +-- VPALACE.DAT.tns
Note : Les fichiers audio (DIGISND*.DAT, IBM_SND*.DAT, MIDISND*.DAT) ne sont pas nécessaires — le son n'est pas supporté sur la Nspire.

Lancer le jeu

Ouvrez l'explorateur de fichiers de votre calculatrice, naviguez vers le dossier contenant prince.tns et lancez-le.

Conseils de jeu

  • Courir : Maintenir la flèche gauche ou droite
  • Saut en courant : Pendant la course, appuyer sur haut
  • Saut sur place : Appuyer sur haut à l'arrêt
  • S'accroupir : Appuyer sur bas
  • Marche prudente : Maintenir Ctrl + appuyer sur une direction
  • Agripper un rebord : Maintenir Ctrl en tombant près d'un rebord
  • Grimper : Maintenir Ctrl en suspension, puis appuyer sur haut
  • Combat : Ctrl pour attaquer, Ctrl pour parer quand l'ennemi frappe
  • Sauvegarde rapide : Appuyer sur Tab pour sauvegarder votre position (niveau, PV, temps)
  • Chargement rapide : Appuyer sur Del pour restaurer votre sauvegarde
Sauvegarde rapide : Le fichier de sauvegarde (QUICKSAVE.SAV.tns) est stocké à côté de l'exécutable. Vous pouvez sauvegarder, quitter le jeu, le relancer plus tard et charger votre position sauvegardée.

Problèmes connus

  • Pas de son (la Nspire n'a pas de haut-parleur)
  • Bref clignottement de couleurs lors des transitions d'écran (cinématiques, écran titre). Le gameplay n'est pas affecté.
