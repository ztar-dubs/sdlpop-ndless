Prince of Persia for TI-Nspire CX

SDLPoP-Ndless — Port of Prince of Persia (1989) for the TI-Nspire CX calculator family.

Installation

Requirements

  • TI-Nspire CX, CX CAS, CX II, or CX II CAS calculator
  • Ndless installed on the calculator
  • Original Prince of Persia DOS game data files (.DAT files)

File structure

Copy the following files to your calculator. The executable and the data/ folder must be in the same directory.

your_folder/ |-- prince.tns (game executable) +-- data/ |-- CDUNGEON.DAT.tns |-- CPALACE.DAT.tns |-- EDUNGEON.DAT.tns |-- EPALACE.DAT.tns |-- FAT.DAT.tns |-- GUARD.DAT.tns |-- GUARD1.DAT.tns |-- GUARD2.DAT.tns |-- KID.DAT.tns |-- LEVELS.DAT.tns |-- PRINCE.DAT.tns |-- PV.DAT.tns |-- SHADOW.DAT.tns |-- SKEL.DAT.tns |-- TITLE.DAT.tns |-- VDUNGEON.DAT.tns |-- VIZIER.DAT.tns +-- VPALACE.DAT.tns
Note: Audio files (DIGISND*.DAT, IBM_SND*.DAT, MIDISND*.DAT) are not needed — audio is not supported on the Nspire.

Launching the game

Open the file browser on your calculator, navigate to the folder containing prince.tns, and launch it.

MenuToggle display: 320x200 (centered) / 320x240 (fullscreen)

Gameplay tips

  • Run: Hold left or right arrow
  • Running jump: While running, press up
  • Standing jump: Press up while standing still
  • Crouch: Press down
  • Careful step: Hold Ctrl + press a direction
  • Grab a ledge: Hold Ctrl while falling near a ledge
  • Climb up: Hold Ctrl while hanging, then press up
  • Fight: Ctrl to attack, Ctrl to parry when the enemy strikes
  • Quick save: Press Tab to save your current position (level, HP, time)
  • Quick load: Press Del to restore your saved position
Quick save: The save file (QUICKSAVE.SAV.tns) is stored next to the executable. You can save, quit the game, restart later and load your saved position.

About this port

Open source project

This port is based on SDLPoP (SDL Prince of Persia), an open-source port of the classic DOS game Prince of Persia (1989) by Jordan Mechner.
